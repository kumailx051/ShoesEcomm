## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/kumailx051/ShoesEcomm.git
```

2. Navigate to the directory:
```bash
cd ShoesEcomm
```

3. Open `index.html` in your web browser or use a local server:
```bash
python -m http.server 8000
# or
npx http-server
```

## Customization

### Colors
Edit the CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #ff6b6b;      /* Main brand color */
    --secondary-color: #1a1a1a;    /* Dark color */
    --accent-color: #4ecdc4;       /* Accent color */
}
```

### Products
Edit the product cards in `index.html` to add your own products with:
- Product name
- Category
- Prices
- Images
- Ratings

### Content
All text content is easily editable in `index.html`. Simply update:
- Company name
- Product information
- Contact details
- Social media links
